Cyber Tech Fest

The event has 2 levels of technical events. Each event has exciting prize money.

High-Level Technical Events:

The events are meant for B.Tech student enthusiasts who are having a good idea about cybersecurity.

CTF (Capture The Flag) (6Hrs) is a competition containing challenges in different domains of cybersecurity such as Forensics, Web Exploitation, Reverse Engineering, OSINT, etc…
Ideathon with Cybersecurity – The theme will be given on the day of the event.
Basic Level Technical Events:

The events help beginner-level B.Tech students to enrich their skills in cybersecurity and lead their way to becoming Cybersecurity professionals.

Scavenger Hunt: Quiz-like competition with clues given in the description. And this will be an individually participating competition. Questions will be based on web exploitation.
Cybertrivia: An oral quiz competition that requires a team to participate in the event.
Debug the Code: An individual event where each participant will be given a code with errors and they should find the errors and correct them.
Identify a Language: An individual event where each participant will be given some program and the participant must identify the language.
Code in an Unknown Language: An individual competition where participants need to learn a new language in less amount of time and write the code for the given problem.
